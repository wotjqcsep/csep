## 프로젝트 개요

### 비전
컴퓨터 A/S 접수부터 출동, 수리, 판매, ERP까지 **하나의 통합 시스템**으로 운영  
- PC와 스마트폰에서 동일한 데이터 사용
- 1인 업체부터 직원이 많은 업체까지 확장 가능

### 핵심 특징
- **접수 다중화**: 전화, SMS, 카카오톡, 직접 등록
- **실시간 동기화**: PC ↔ 스마트폰 양방향 동기화
- **역할 기반**: 대표/기사별 다른 화면과 기능
- **단계적 확장**: MVP → 기능 확장 → ERP 통합

## 1. 접수 시스템 (가장 중요)

### 일반 고객 접수
- **전화**: 전화번호 자동 인식 → 고객 자동 검색 → 신규면 자동 등록 → 접수 생성
- **문자(SMS)**: 자동 고객 조회 → 접수 생성
- **카카오톡**: 범위 내에서 고객 확인 → 접수 등록
- **직접 등록**: 관리자가 수동으로 입력

### 컴닥터(협력업체) 접수
- 전화, 문자, 카카오톡으로 접수
- 특정 번호 또는 규칙으로 자동 인식
- 일반 고객과 별도 분류 관리
- 향후 다른 협력업체도 쉽게 추가 가능

## 2. 전화 접수 프로세스

```
전화 수신
  ↓
전화번호 자동 인식
  ↓
고객 자동 검색
  ├─ 기존 고객: 정보 표시
  └─ 신규 고객: 자동 등록
  ↓
접수 생성
  ↓
화면에 표시:
  - 고객명
  - 주소
  - 보유 컴퓨터
  - 판매내역
  - 수리내역
  - 메모
  - 미수금
```

## 10. 컴퓨터 관리

각 컴퓨터마다 상세 정보 관리:
- **하드웨어**: CPU, RAM, SSD, HDD, 메인보드
- **운영체제**: Windows 버전
- **소프트웨어**: Office, 백신, 기타 프로그램
- **네트워크**: IP, MAC Address
- **기타**: 프린터, 모니터, 시리얼번호
- **거래 정보**: 판매일, 보증기간, 조립여부
