## 1. Backend (서버 언어 & 프레임워크)

### 옵션 1: Python + Flask/FastAPI

| 항목 | 설명 |
|------|------|
| **장점** | - 개발 빠름<br>- 초보자도 배우기 쉬움<br>- 라이브러리 풍부<br>- 현재 경험 있음 (field-service) |
| **단점** | - 성능이 상대적으로 낮음<br>- 대규모 동시 접속 약할 수 있음 |
| **추천** | ⭐⭐⭐⭐⭐ (field-service 경험 있으면 가장 빠름) |
| **예시** | `FastAPI` (비동기 지원, 실시간 기능에 좋음) |
| **학습시간** | 적음 (이미 Python 숙련) |

**선택 시:**
```
Language: Python 3.10+
Framework: FastAPI (또는 Flask)
DB Driver: psycopg2 (PostgreSQL)
Async: asyncio + aiohttp
Real-time: WebSocket (FastAPI 내장)
```

---

### 옵션 2: Node.js + Express/NestJS

| 항목 | 설명 |
|------|------|
| **장점** | - 빠른 성능<br>- 실시간 처리 좋음 (이벤트 기반)<br>- JavaScript 하나로 Frontend/Backend 통일 가능<br>- 동시 접속 처리 탁월 |
| **단점** | - JavaScript 학습 필요<br>- 타입 안정성 문제 (TypeScript 권장) |
| **추천** | ⭐⭐⭐⭐ (새로 배워도 괜찮으면 좋은 선택) |
| **예시** | `Express` (간단) 또는 `NestJS` (구조적) |
| **학습시간** | 중간 (새로 배워야 함) |

**선택 시:**
```
Language: JavaScript (TypeScript 권장)
Framework: Express 또는 NestJS
DB: TypeORM + PostgreSQL
Real-time: Socket.io
Package Manager: npm 또는 yarn
```

### ✅ 현 상황에서 추천

**WebSocket + Socket.io**

구현:
```python
# FastAPI
from fastapi import WebSocket
from fastapi.middleware.cors import CORSMiddleware
import socketio

sio = socketio.AsyncServer(async_mode='asgi', cors_allowed_origins='*')

@sio.event
async def reception_update(data):
    # 모든 클라이언트에 브로드캐스트
    await sio.emit('reception_updated', data, to=None)
```

```javascript
// Frontend & Mobile
import io from 'socket.io-client'

const socket = io('http://server-url')

socket.on('reception_updated', (data) => {
    console.log('새 접수:', data)
})
```
