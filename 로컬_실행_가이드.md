## 🚀 Backend (FastAPI) 실행

### Step 1: 가상환경 생성

```powershell
cd E:\dev\CSEP\backend

# 가상환경 생성
python -m venv venv

# 가상환경 활성화 (Windows PowerShell)
venv\Scripts\Activate.ps1

# 또는 CMD라면
venv\Scripts\activate.bat
```

**활성화되면 프롬프트 앞에 `(venv)`가 표시됨**

### Step 2: 패키지 설치

```powershell
# 가상환경이 활성화된 상태에서
pip install -r requirements.txt
```

### Step 3: 서버 실행

```powershell
python -m uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

**성공하면:**
```
Uvicorn running on http://127.0.0.1:8000
Application startup complete
```

### Step 4: API 테스트

**브라우저에서:**
```
http://localhost:8000/docs
```

또는 **PowerShell에서:**
```powershell
# 고객 조회
curl http://127.0.0.1:8000/api/customers

# 접수 조회
curl http://127.0.0.1:8000/api/receptions

# 기사 조회
curl http://127.0.0.1:8000/api/engineers

# 대시보드
curl http://127.0.0.1:8000/api/dashboard
```

## 🗄️ Database (PostgreSQL) 설정 (선택)

현재는 **더미 데이터**를 사용하고 있어서 PostgreSQL이 필수는 아닙니다.

나중에 실제 DB가 필요하면:

### Docker로 실행 (권장)

```powershell
docker run -d \
  -e POSTGRES_PASSWORD=password \
  -e POSTGRES_DB=csep \
  -p 5432:5432 \
  postgres:15
```

### 또는 PostgreSQL 설치

```
https://www.postgresql.org/download/
```

## 📊 현재 API 엔드포인트

### 고객 관리
```
GET    /api/customers              전체 고객 조회
GET    /api/customers/{id}         고객 상세 조회
POST   /api/customers              새 고객 추가
```

### 접수 관리
```
GET    /api/receptions             전체 접수 조회
GET    /api/receptions?status=new  상태별 접수 조회
POST   /api/receptions             새 접수 생성
PUT    /api/receptions/{id}/assign 기사 배정
PUT    /api/receptions/{id}/status 상태 업데이트
```

### 기사 관리
```
GET    /api/engineers              전체 기사 조회
GET    /api/engineers/{id}         기사 상세 조회
PUT    /api/engineers/{id}/status  상태 업데이트 (idle, working, on_the_way, off_duty)
PUT    /api/engineers/{id}/location 위치 업데이트
```

### 대시보드
```
GET    /api/dashboard              통계 및 현황
```

### 헬스체크
```
GET    /health                     서버 상태
```

## 🧪 테스트 예제

### PowerShell에서 고객 추가

```powershell
$body = @{
    name = "김철수"
    phone = "01099999999"
    address = "강남구 역삼동"
} | ConvertTo-Json

curl -X POST `
  -H "Content-Type: application/json" `
  -d $body `
  http://127.0.0.1:8000/api/customers
```

### PowerShell에서 접수 추가

```powershell
$body = @{
    customer_id = 1
    symptom = "데스크톱 전원 안 켜짐"
    channel = "phone"
} | ConvertTo-Json

curl -X POST `
  -H "Content-Type: application/json" `
  -d $body `
  http://127.0.0.1:8000/api/receptions
```

### PowerShell에서 기사 배정

```powershell
curl -X PUT `
  http://127.0.0.1:8000/api/receptions/1/assign?engineer_id=1
```

## 🐛 문제 해결

### "포트 8000이 이미 사용 중"

```powershell
# 다른 포트 사용
python -m uvicorn main:app --reload --host 127.0.0.1 --port 8001
```

### "ModuleNotFoundError: No module named 'fastapi'"

```powershell
# 가상환경이 활성화되지 않았음
venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

### "CORS 에러"

현재 CORS가 모든 출처 허용(`*`)으로 설정되어 있으므로 문제없어야 함.
